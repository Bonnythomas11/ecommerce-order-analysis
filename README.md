# E-commerce Order Analysis using SQL

## 📌 Objective
Analyze order lifecycle data to identify delivery success rates and key drop-off points in the e-commerce process.

## 🛠 Tools Used
- MySQL
- DBeaver

## 📊 Dataset
Brazilian E-commerce Public Dataset (Olist)

## 🔍 Key Analysis

### 1. Order Distribution by Status
- Delivered orders form the majority
- Significant number of orders are cancelled or unavailable

### 2. Delivery Success Rate
- Calculated percentage of successfully delivered orders

### 3. Customer Experience Insight
- Linked order status with review scores

## 💡 Key Insights
- ~60–70% orders are successfully delivered
- ~15–20% orders fail (cancelled + unavailable)
- Major drop-off occurs before final delivery stage

## 🚀 Business Recommendations
- Improve supply chain efficiency
- Reduce cancellation rates
- Address inventory availability issues
